## 重写过程INSERTION-SORT，使之按非升序（而不是非降序）排序

```
  INSERTION-SORT(A)
    for j = 2 to A.length
        key = A[j]
        i = j - 1
        while i > 0 and A[i] < key
            A[i + 1] = A[i]
            i = i - 1
        A[i + 1] = key
 ```
